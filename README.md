# React Hooks ⚓

**NOTE:** Crestron related hooks have been deprecated. [Learn more here](DEPRECATED.md).

A collection of useful hooks ⚓ for React

## Installation :zap:

```sh
npm install @norgate-av/react-hooks

# or

yarn add @norgate-av/react-hooks
```
